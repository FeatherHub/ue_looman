void ASDashProj::BeginPlay() 
{
  Super::BeginPlay();
  GetWorldTimeManager().SetTimer(
    TimerHandle_DelayedDetonate,
    this,
    &ASProj::Explode,
    DetonateDelay
  );
}

---
api design
  GetWorldTimeManager returns a reference not a pointer

assumption
  timer manager is always available

pattern
  api communicates intention on the type system

mechanism
  returning a pointer
    imply non-existence (nullptr)
  returning a reference
    ensure the existence
    no null check needed
